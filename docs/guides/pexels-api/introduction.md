# Introduction

来源：`https://www.pexels.com/api/documentation/#introduction`

Pexels API 允许以编程方式访问 Pexels 的内容库（照片与视频）。API 为 RESTful JSON，可用任意语言/框架的 HTTP 客户端进行调用，也可使用官方客户端库。

如需帮助与排障，可访问 Help Center：`https://help.pexels.com/hc/en-us/categories/900001326143-API`

## Base URL 说明（重要）

- **照片相关接口**：以 `https://api.pexels.com/v1/` 开头
- **视频相关接口**：以 `https://api.pexels.com/videos/` 开头
