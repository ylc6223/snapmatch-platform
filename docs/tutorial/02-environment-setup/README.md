# 第二阶段：环境搭建与配置

> **学习目标**：搭建本地开发环境，配置云服务  
> **预计时长**：3-4 小时  
> **难度等级**：⭐⭐☆☆☆

## 本阶段内容

### 1. [本地开发环境搭建](./01-local-dev-environment.md)

**大纲**：

- Node.js 安装与版本管理（nvm）
- pnpm 包管理器安装
- MySQL 数据库安装与配置
  - 本地 MySQL
  - Docker MySQL
- VS Code 开发环境配置
  - 推荐插件
  - 代码片段配置
- 环境变量配置

**学习成果**：搭建完整的本地开发环境

---

### 2. [云服务配置](./02-cloud-services-setup.md)

**大纲**：

- CloudBase 环境创建
- MySQL 数据库配置
  - 数据库实例创建
  - 连接配置
  - 安全组设置
- Cloudflare R2 配置
  - Bucket 创建
  - API Token 生成
  - 自定义域名配置
- 环境变量管理（开发/生产）

**学习成果**：掌握云服务配置流程

---

### 3. [开发工具配置](./03-development-tools.md)

**大纲**：

- Git 配置与使用
  - 基础命令
  - .gitignore 规则
- ESLint 与 Prettier 配置
  - 代码规范
  - 自动格式化
- Husky 与 lint-staged（提交前检查）
- 调试工具配置
  - Chrome DevTools
  - VS Code 调试配置

**学习成果**：配置高效的开发工具链

---

### 4. [团队协作规范](./04-team-collaboration.md)

**大纲**：

- Git Flow 工作流
  - 分支管理策略
  - 提交信息规范
- Code Review 流程
  - Review 清单
  - 最佳实践
- 文档编写规范
- 问题排查指南

**学习成果**：建立规范的团队协作流程
